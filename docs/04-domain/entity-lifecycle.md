# Entity Lifecycle

最后更新：2026-03-31

## 1. Project 生命周期

创建：

- 从工作台新建
- 从阵型快捷入口新建
- 从项目复制新建副本

变化：

- 在编辑器中被持续修改
- 由自动保存草稿承接临时进度

结束：

- 保留在项目列表中
- 或被用户删除
- 删除后从项目索引与最近项目索引中同步移除

## 2. LocalDraft 生命周期

创建：

- 编辑器自动保存时生成

变化：

- 随项目修改被覆盖更新

结束：

- 成功恢复并继续编辑
- 被更新的自动保存覆盖
- 被用户跳过恢复
- 在所属项目被删除后同步清理

## 3. FormationTemplate 生命周期

创建：

- 内置提供
- 用户保存自定义阵型

变化：

- 用户后续继续覆盖调整

结束：

- 被用户删除自定义模板

## 4. ReferenceImage 生命周期

创建：

- 用户导入图片

变化：

- 进入项目并被复制到应用本地存储

结束：

- 项目删除后可随项目清理

## 5. Export 生命周期

创建：

- 用户触发 PNG 或 GIF 导出

变化：

- 仅生成导出结果，不应修改项目实体

结束：

- 导出完成
- 或导出失败并给出提示

## 6. ProjectIndexEntry 生命周期

创建：

- 项目创建时加入索引
- 项目首次打开或被复制后加入最近项目索引

变化：

- 随项目重命名和更新时间变化同步更新

结束：

- 项目删除后移除
- 索引失效时允许单独清理异常条目

## 7. MatchMeta 生命周期

创建：

- 随 Project 创建而初始化

变化：

- 在编辑器页面级属性中被修改
- 随 Project 保存与自动保存一起持久化

结束：

- 随 Project 删除而删除

## 8. ExportConfig 生命周期

创建：

- 用户打开导出面板或触发导出时临时构建

变化：

- 随导出格式、比例和显示开关选择变化

结束：

- 导出完成后丢弃
- 用户取消导出后丢弃
- 不跨会话保留
