# Roadmap

## 1. Purpose
This document captures the likely sequencing after V1 without redefining the current release.

## 2. V1

Focus:
- stable core editor
- asset management
- share/view flows
- workspace integrity
- responsive usability

## 3. Post-V1 Candidates

### 3.1 Collaboration expansion
- richer permission controls
- team collaboration enhancements
- comment/review workflows

### 3.2 Content output expansion
- stronger export presets
- richer animated output
- presentation refinements

### 3.3 AI-assisted expression
- title generation
- summary generation
- script/caption assistance

AI remains post-V1 and should not block the current release.

### 3.4 Data/analysis expansion
- richer review metadata
- analytics overlays
- deeper match-recap tooling

## 4. Roadmap Principle

Post-V1 work should prioritize:
1. product reliability
2. high-frequency user workflows
3. expression and reuse
4. only then advanced intelligence/features
