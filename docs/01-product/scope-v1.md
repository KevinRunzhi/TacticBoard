# V1 Scope

最后更新：2026-03-30

## 1. V1 定位

V1 是一个面向业余足球爱好者的跨端本地战术板应用首发版本。

核心特征：

- `Windows + Android` 首发
- `离线单机`
- `免注册`
- 本地保存
- PNG 导出
- Windows 首发优先支持基础 GIF 导出

## 2. V1 核心目标

- 让用户可以快速开始一张战术板创作
- 让用户可以继续编辑最近项目
- 让用户可以在桌面端和 Android 端都完成完整创作闭环
- 让用户可以稳定导出图片，Windows 端可导出基础 GIF
- 让产品在不依赖账号和网络的条件下完成主要使用场景

## 3. V1 主链路

`工作台 -> 新建项目/打开最近项目 -> 编辑器 -> 本地保存 -> PNG / Windows GIF 导出`

## 4. V1 页面范围

保留：

- 工作台 / 首页
- 项目页
- 编辑器页
- 设置页
- 404 页面

不保留：

- 登录页
- 注册页
- 绑定账号页
- 在线分享页
- 球队页
- 独立模板页
- 营销落地页

## 5. V1 功能范围

纳入：

- 场地模板
- 阵型模板与自定义阵型保存
- 球员、足球、文本、区域、参考底图
- 基础线路系统
- 步骤帧与播放
- 比赛信息（比分、分钟、回合阶段）与标题区
- 本地自动保存与手动保存
- 最近项目、项目复制、项目删除
- 撤销 / 重做、多选、对齐 / 吸附、锁定 / 隐藏
- 双球员展示样式、成套主题、主客队换肤、文本 / 线路 / 区域风格预设
- PNG 导出
- Android 图片导出后的系统分享
- Windows 基础 GIF 导出

不纳入：

- 账号体系
- 云同步
- 在线分享链接
- 项目文件导入导出
- 本地备份包
- 历史版本
- 模板库
- 快捷键体系
- 批量操作
- 复杂图层 / 分组
- 打印增强
- 球员职责卡
- 战术对比模式
- Android GIF 导出
- MP4 导出
- 内容创作包装、社媒适配、品牌化导出
- AI 文案辅助

说明：

- 更完整的第一阶段不纳入清单和限制条件，以 `docs/football-tactics-board-requirements.md` 第 17 节为准

## 6. V1 平台边界

Windows：

- 优先承载长时间编辑体验
- 优先承载 GIF 导出

Android：

- 承担完整编辑主链路
- 优先保证触控操作、PNG 导出和系统分享稳定
- 首发不强制承诺 GIF 导出
- 首发以 GitHub Releases 下载 APK、手动安装和手动更新为主
- 当前目标支持 Android 10 及以上
- Android 阶段细化与关闭条件以 `docs/05-engineering/android-packaging-plan.md` 和 `docs/android-packaging/` 为准

## 7. V1 交付标准

只有同时满足以下条件，V1 才算可交付：

- 主页面结构稳定
- 核心编辑闭环打通
- 离线可用
- 本地保存可恢复
- PNG 导出稳定
- Windows GIF 导出可用
- 关键异常路径有兜底提示
