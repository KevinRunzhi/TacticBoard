# Roadmap

最后更新：2026-03-31

## 1. 当前阶段

当前处于“V1 文档基线建立”阶段。

目标：

- 冻结主文档
- 重建 01-07 派生文档
- 为前端改造和功能开发提供统一基线

## 2. 阶段划分

### Phase 0：文档与框架冻结

- 冻结 PRD、requirements、信息架构
- 重建 01-07 分层文档
- 冻结页面结构和壳层边界

### Phase 1：页面框架与本地项目主链路

- 工作台
- 项目页
- 设置页
- 编辑器壳层
- 本地保存 / 最近项目 / 项目复制 / 删除

### Phase 2：编辑器核心能力

- 场地与阵型
- 球员 / 足球 / 文本 / 区域 / 参考底图
- 线路系统
- 步骤帧与播放
- 基础查看与缩放

### Phase 3：导出与质量收口

- PNG 导出
- Windows GIF 导出
- Android 图片导出后的系统分享
- 异常提示与失败兜底
- 验收清单和测试矩阵收口

### Phase 4：后续增强评估

- 项目文件与备份包
- 模板库
- 快捷键与批量操作
- 复杂图层
- 打印增强
- 战术对比
- Android GIF

## 3. 当前建议

- 当前阶段不要再扩张产品范围
- 所有新增能力先看是否影响 V1 主链路
- 没有进入主链路的能力默认进入后续评估
