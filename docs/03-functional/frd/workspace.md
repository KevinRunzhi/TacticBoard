# Workspace FRD

最后更新：2026-03-30
页面：`/`

## 1. 页面目标

- 提供新建项目的最快入口
- 提供最近项目恢复入口
- 提供草稿恢复提示

## 2. 必备模块

- 新建项目入口
- 继续上次编辑
- 最近项目列表
- 内置阵型快捷开始
- 草稿恢复提示

## 3. 模块规则

### 新建项目入口

- 点击后直接进入编辑器
- 不经过登录、模板页或其他平台化入口

### 最近项目列表

- 按最近编辑时间排序
- 点击后进入对应项目编辑器
- 不暴露分享、团队或账号相关操作

### 草稿恢复提示

- 仅在存在可恢复草稿时显示
- 允许恢复或跳过

## 4. 空状态

- 无最近项目时，首屏强调新建项目
- 无草稿时，不显示恢复提示

## 5. 异常状态

- 最近项目失效时允许移除条目
- 草稿损坏时允许跳过恢复
