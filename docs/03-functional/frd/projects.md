# Projects FRD

最后更新：2026-03-30
页面：`/projects`

## 1. 页面目标

- 管理本地项目
- 为项目复制、重命名、删除提供集中入口

## 2. 必备模块

- 项目列表
- 默认按最近编辑时间倒序排序
- 项目操作菜单

## 3. 支持操作

- 打开项目
- 重命名
- 复制
- 删除

## 4. 规则

- 复制后的项目默认生成可区分名称
- 删除前必须二次确认
- 第一阶段不做标签、收藏、复杂检索
- 第一阶段不在项目页提供分享、团队、模板库或账号相关操作

## 5. 空状态

- 没有项目时，引导新建第一个项目

## 6. 异常状态

- 项目记录失效时，提示用户移除无效条目
