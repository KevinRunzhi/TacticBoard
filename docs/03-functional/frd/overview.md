# FRD Overview

最后更新：2026-03-30

## 1. 页面范围

V1 页面级功能规格只覆盖：

- 工作台
- 项目页
- 编辑器页
- 设置页
- 404 页面

## 2. 共享规则

- 所有页面默认离线可用
- 所有页面默认不依赖账号
- 所有页面默认不依赖远程拉取
- 进入编辑器与打开最近项目不得被更新检查阻塞

## 3. 共享对象

页面级规格围绕以下核心对象展开：

- 本地项目
- 自动保存草稿
- 自定义阵型模板
- 导出结果

## 4. 页面间职责分工

- 工作台：开始与恢复
- 项目页：整理与进入
- 编辑器页：创作与导出
- 设置页：偏好与说明

## 5. 页面间不应混淆的职责

- 工作台不承担复杂项目管理
- 项目页不承担创作
- 编辑器页不承担模板库和账号流程
- 设置页不承担资产管理
